Bem vindo ao projeto de teste da Oliveira Trust

Este projeto foi desenvolvido como teste para a vaga de desenvolvedor na Oliveira Trust. O projeto foi desenvolvido em Angular 11. O objetivo dele é criar um Dashboard bonito e funcional.

Os dados apresentados foram criados utilizando o Mockaroo.
